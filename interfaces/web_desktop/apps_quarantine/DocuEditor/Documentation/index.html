FriendUP Documentation Editor

Introduction

Welcome to the documentation editor in FriendUP. Please refer to this document if you are stuck or need information on how to make the best of this application.

What is a documentation editor?

The documentation editor is where you write how things work. In this editor, you can edit your own application's documentation, as well as review and comment on the other FriendUP API's and application documents.

How to use Documentation Editor

When you start Documentation Editor, you have the top toolbar, where you can search and filter the topics list. Then below, you have the scroll pane, where you can see all the existing topics sorted or filtered by status. On the bottom you see the bottom toolbar, where you can add a topic.

When you have a topic to add, click on the "Add topic" button on the bottom toolbar. This will bring up a new view where you can describe your topic. Once you have filled in all the form fields there, you can click on "Save topic". This will submit your topic to an administrator for review (if applicable).
